[202601 abcd12345678 一级]累加特定数字题解

2026-03-22 1 min read # 题解

#题目大意
给你两个正整数a,b,要你求a到b之间的偶数之和

#解题思路
若a>b,交换a,b。遍历a到b，判断i是否为偶数,是则让ans+i

#核心代码

#include<bits/stdc++.h>
using namespace std;
int main(){
    int a,b;
    int ans=0;
    cin>>a>>b;
    if(a>b)swap(a,b);
    for(int i=a;i<=b;i++){
        if(i%2==0)ans+=i;
    }
    cout<<ans;
    return 0;
}
