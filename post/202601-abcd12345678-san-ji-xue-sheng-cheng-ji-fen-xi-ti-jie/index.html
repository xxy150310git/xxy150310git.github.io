[202601 abcd12345678 三级]学生成绩分析题解

2026-03-22 2 min read # 题解

#题目大意
给你一些学生的分数，要你求他们的平均分、最低分、最高分、高于平均分人数。

#解题思路
1.求平均分：累加学生分数得到总分，再用总分除以人数。

2.求最大最小值：每次输入将max与分数取最大值，min与分数取最小值。

3.高于平均分人数：遍历学生分数，将学生分数与平均分比较，若高于平均分则让高于平均分人数++。

#核心代码


#include<iostream>
#include<algorithm>
#include<iomanip>
//考试别用万能头
using namespace std;
int a[105];//学生成绩
int main(){
    //freopen("---.in","r",stdin);
    //freopen("---.out","w",stdout);
    //不写freopen见祖宗
    int n,ma=0,mi=101,h=0;
    cin>>n;
    for(int i=1;i<=n;i++){
        cin>>a[i];
        h+=a[i];//求和
        ma=max(ma,a[i]);
        mi=min(mi,a[i]);
    }
    double av=h*1.0/n;//平均数=数列之和÷数列长度
    int r=0;
    for(int i=1;i<=n;i++){
        if(a[i]>av)r++;//计算大于平均分人数
    }
    cout<<"Average: "<<fixed<<setprecision(1)<<h*1.0/n<<endl;
    cout<<"Max: "<<ma<<", Min: "<<mi<<endl;
    cout<<"Above average: "<<r;
    return 0;//好习惯
}
